## 프로젝트 소개
- 컨셉
  - OOTD란 ‘오늘 입은 옷차림’, ‘오늘의 패션’(Outfit Of The Day)의 준말로,
    당일 또는 특정 상황에서 입은 자신의 옷차림을 촬영하여 소셜미디어 등에 업로드하는 행위를 가리킨다는 뜻으로
    ‘HOW ABOUT OOTD’는 사용자에게 다양한 브랜드와 데일리룩을 추천하고 의류를 판매하는 쇼핑몰
- 프로젝트 인원: 2명
- 프로젝트 기간: 2022.05.27(금) ~ 2022.07.25(월) / 2달
- 역할: 프론트엔드 및 UI 개발
